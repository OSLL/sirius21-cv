Задача:
Используя представленный шаблон кода написать в отмеченном месте блок, проверяющий существование файла по указаному пути.
При запуске программы должно происходить следующее:

Целевое поведение программы
Если путь, указанный в переменной path ведет к существующему файлу позволить программе выполнить print(), стоящий в конце программы

path = "README.MD"

File exists. Path OK
Если путь неверный, то программа должна вернуть ошибку, сказать что путь неверный и завершить выполнение программы.

path = "abcd"

Invalid file path. Type correct path
Обрати внимание, print("File exists. Path OK") в программе уже есть. Его добавлять не надо.

После написания программы надо будет запросить пулл-реквест на объединение решения с главной веткой разработки.
