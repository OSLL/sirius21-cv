Напишите программу, используя шаблон https://github.com/OSLL/sirius21-cv/blob/main/main.py, которая:

Получает в качестве аргумента командной строки путь к файлу-изображению
Программа должна проверять существование файла по переданному пути
В случае, если файла не существует, необходимо вывести соответствующее сообщение и завершить выполнение программы
Проверка существования файла должна быть реализована в отдельной функции
Открывает этот файл с помощью библиотеки CV (вынесите данную функциональность в отдельную функцию)
Получает маску для цветового диапазона (цветовой диапазон выберите сами)
Накладывает маску на исходное изображение
Сохраняет полученное изображение в файл (имя нового файла выберите сами)
Имя сохраняемого может задаваться вторым аргументом командной строки (прим.: может быть не указано, тогда следует использовать некоторое название по умолчанию)
Разместите полученный файл в репозитории в папке с названием Фамилия_ИО латиницей (например, Ivanov_DV)
